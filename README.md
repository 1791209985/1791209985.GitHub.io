### **打开终端的方式**

1.通过点击左侧工具栏中的终端图标打开

2.通过组合键 ctrl + alt + t 打开一个新的终端

3.分屏打开同路径的终端  ctrl + shift + t

4.打开一个同路径的新终端 ctrl + shift +n

### **命令行提示符**

linux@hqyj:~$

linux：当前用户

hyqj：主机名

~：当前路径   ~ 表示当前在家目录

$：权限  (linux的用户分为管理员用户和普通用户)

管理员用户有当前系统中所有的权限

普通用户只在自己的家目录下有操作权限

> ​		修改文件权限



| 当前用户 | 当前用户同组人 | 其他人 |
| :------: | :------------: | :----: |
|   user   |     group      | other  |
|    u     |       g        |   o    |

|      | 权限 | 8进制 | 二进制 |
| :--: | :--: | :---: | :----: |
|  x   | 执行 |   1   |  001   |
|  w   |  写  |   2   |  010   |
|  r   |  读  |   4   |  100   |

​											一位八进制等同三位二进制

| 0    | 1    | 2    | 3    | 4    | 5    | 6    | 7    |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 000  | 001  | 010  | 011  | 100  | 101  | 110  | 111  |

​									

​									一个1位的八进制整数		一个权限

​									一个3位的八进制整数		一个文件三种人的权限

​												**000**			ugo都没有任何权限

​												**123**			u  x   ；		 g  w  ；		o   w和x；
