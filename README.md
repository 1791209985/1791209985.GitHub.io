 **打开终端的方式**

1.通过点击左侧工具栏中的终端图标打开

2.通过组合键 ctrl + alt + t 打开一个新的终端

3.分屏打开同路径的终端  ctrl + shift + t

4.打开一个同路径的新终端 ctrl + shift +n

### **命令行提示符**

linux@hqyj:~$

linux：当前用户

hyqj：主机名

~：当前路径   ~ 表示当前在家目录

$：权限  (linux的用户分为管理员用户和普通用户)

#### 					预编译：

预编译是做些代码文本的替换工作。

处理以# 开头的指令 , 比如拷贝 #include 包含的文件代码，#define 宏定义的替换 ， 条件编译等，就是为编译做的预备工作的阶段。

主要处理#开始的预编译指令，预编译指令指示了在程序正式编译前就由编译器进行的操作，可以放在程序中的任何位置。

C 编译系统在对程序进行通常的编译之前，首先进行预处理  。
